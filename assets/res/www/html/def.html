이벤트 사용법

설명

태그의 속성으로 페이지 이동 이벤트를 사용할수 있다.

data-event-page='empty.html'

태그의 속성으로 추가하고 이동할 페이지를 넣어 준다.


data-event-animat='left'

태그의 속성으로 추가하고 이동할 페이지를 넣어 준다. 속성은 다음과 같다.

"def":"DEFALUT",
"none":"NONE",
"left":"SLIDE_LEFT",
"right":"SLIDE_RIGHT",
"top":"SLIDE_TOP",
"bottom":"SLIDE_BOTTOM",
"in":"ZOOM_IN",
"out":"ZOOM_OUT",
"fade":"FADE",
"up":"MODAL_UP",
"down":"MODAL_DOWN",
